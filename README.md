# dsctest

dsctest is a repo containing all the code that was used to conduct DSC-RIT recruitment tests.

## Installation

Use git or download zip.

```git
git clone https://github.com/nimishbongale/dsctest.git
```

## Usage

Run the server with 

```python3
python3 server.py
```
